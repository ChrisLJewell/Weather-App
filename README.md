# Weather-App
A responsive weather application built using Javascript and OpenWeather API.

## Features
- Search by city
- Search by city and state (US only)
- Current weather 
- 16-day forecast
- Fahrenheit/Celsius toggle
- Geolocation support

## Technologies 
- HTML
- CSS
- Javascript
- OpenWeather API


## What I learned
- Working with REST APIs
- Handing asynchronous Javascipt using fetch
- HTML DOM Document createElement()
- Geolocation- watchPosition and getPosition
