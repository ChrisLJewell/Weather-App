
# Weather App 🌤️

A responsive weather application built with JavaScript and the OpenWeather API.

---

## Live Demo

https://chrisljewell.github.io/Weather-App/
---

## Features

- Search weather by city
- Search by city and state (US only)
- Current weather conditions
- 16-day forecast
- Fahrenheit / Celsius toggle
- Geolocation support (auto-detect location)

---

## Technologies Used

- HTML
- CSS
- JavaScript (ES6+)
- OpenWeather API
- Figma for weather icon 
- Data visualization with Chart.js (implemented during development but later removed to simplify the UI)

---

## What I Learned

- Working with REST APIs
- Handling asynchronous JavaScript using fetch and async/await
- DOM manipulation (creating and updating elements dynamically)
- Using geolocation with `getCurrentPosition`
- Clearing and re-rendering UI data properly
- Basic data visualization with Chart.js

---


## Future Improvements

- Improve mobile layout and responsiveness
- Add loading states while fetching data
- Improve UI spacing and visual hierarchy
- Save recent searched cities
- Improve toggle logic
